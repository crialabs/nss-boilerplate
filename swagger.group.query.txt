query
Query operations with a CRAFTS API



GET
/apis/{apiId}/query
Send a parametrized query as configured in an API


This operation needs to reference a valid query template (parameter id) in the API. Note that a query template may define parameters (and some MAY BE REQUIRED). These parameters can be provided as query parameters in the URL
